How to use:
1. Unzip the downloaded archive with software like 7-Zip.
The archive didn't get downloaded? Download it manually.

2. If you are on Windows, run run.bat.
If you are on Linux, run run.sh.
It should set up Python and it's dependencies for Fleasion.

3. Enjoy!
