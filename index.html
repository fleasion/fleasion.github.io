FAQ Download



Frequently asked questions:

▶ Hiding my arms seems like an advantage. Is this cheating?
No. Fleasion isn't a cheat and every built-in
customization option is allowed by StyLiS developers.

▶ Is Fleasion a virus?
No. Fleasion is a completely safe, open-source
one of a kind program for Roblox customization.
It isn't bundled with adware and/or malware

▶ I ran the run.bat file and it instantly closes, what do i do?
Read this guide on manually setting up Fleasion.

▶ I don't see the run.bat file.. Where is it in the archive?
The run.bat file is a batch script, which has a command prompt icon.
You might have file extensions hidden, so you might not see the .bat.

▶ Does Fleasion work with Bloxstrap?
Yes. The "default skyboxes" feature even helps Bloxstrap with custom skyboxes.
If using said feature responds with Bloxstrap not being present,
you either have it in a custom directory or don't have it at all.

▶ How do i get custom skyboxes?
Find some on the internet!

We only support them with Bloxstrap since it's the superior way of using custom skyboxes.
When you find what you're satisfied with, click the "Windows key + R" shortcut, copy
cmd /c "mkdir "%userprofile%\AppData\Local\Bloxstrap\Modifications\PlatformContent\pc\textures\sky" & explorer "%userprofile%\AppData\Local\Bloxstrap\Modifications\PlatformContent\pc\textures\sky""
and press enter. This will create and open the folder where you put your custom skyboxes.

▶ My custom skyboxes aren't working!
If you're on a night time map, custom skyboxes will not work.
Use the 'default skyboxes' option after playing a bit
to have it activate on the maps you already played.

▶ It said that it didn't find the hash. What do I do?
Some actions will require you to do the following to load necessary cache:
- Sights will require you to equip the sight on any gun for it to load
- In-game sounds will require you to trigger the sound (like, shoot the gun)
- Skins will require you to equip the skin or load the skin's preview in the shop
- Default skyboxes require you to play on maps to load the skyboxes first.
- Removing textures will require you to play on maps to load the textures first.
You might also just need to rejoin the Texture Game to reload the assets.

▶ Why are my sleeves sometimes red and sometimes blue?
Phantom Forces applies a tint depending on which team you're playing as.
This isn't possible to remove.

▶ Can we add something that doesn't exist in the game yet?
No. Fleasion can only replace or remove, not create.

▶ Can we have a hash for '...'?
Yes. Request it in our Discord server!
Important, there will be situations where that isn't remotely possible.

▶ Can we get this sound/texture/etc.?
Yes. Request it in our Discord server!

▶ Can we manually edit presets?
Yes. The presets.json file already comes with 1 preset,
comply with JSON formatting while editing the file.

▶ Is it possible to make texture packs?
Yes, with a few caveates.
You will get artifacts and things will not render at range.
